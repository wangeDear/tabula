# Tabula - Chrome 标签页管理器

一个功能强大的 Chrome 扩展，用于管理标签页和保存浏览会话。

## 功能特点

### 🚀 核心功能
- **会话管理**: 保存和恢复浏览会话
- **标签页分组**: 按窗口组织标签页
- **标题编辑器**: 修改标签页标题，支持批量操作
- **快速搜索**: 搜索标签页标题和 URL
- **一键操作**: 快速打开、关闭、切换标签页

### 💫 高级功能
- **完整页面管理**: 点击插件图标打开完整管理页面
- **智能排序**: 按日期、名称、标签数排序
- **批量操作**: 保存所有窗口为会话，批量修改标题
- **右键菜单**: 快速保存当前页面或窗口
- **键盘快捷键**: 快速访问常用功能

### 🎨 界面设计
- **现代化设计**: 深色主题，美观易用
- **响应式布局**: 适配不同屏幕尺寸
- **直观操作**: 简洁的用户界面

## 安装方法

### 开发者模式安装
1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

### 准备图标文件
在 `icons/` 目录下放置以下图标文件：
- `icon16.png` (16x16 像素)
- `icon32.png` (32x32 像素)
- `icon48.png` (48x48 像素)
- `icon128.png` (128x128 像素)

## 使用指南

### 基本操作
1. **点击扩展图标** 打开 Tabula 完整管理页面
2. **浏览会话** 在左侧导航中选择"保存的会话"
3. **创建会话** 点击"新建会话"按钮
4. **管理窗口** 在"打开的窗口"中查看当前标签页
5. **编辑标题** 在"标题编辑器"中修改标签页标题

### 快捷键
- `Ctrl+K` (或 `Cmd+K`): 聚焦搜索框
- 右键菜单: 快速保存页面或窗口

### 搜索功能
- 支持搜索标签页标题
- 支持搜索 URL
- 实时过滤结果
